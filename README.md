# ATM Simulator

A simple ATM simulator built using HTML, CSS and JavaScript.  
This project allows users to log in using a PIN and perform basic banking operations like checking balance, depositing money, withdrawing money, and generating a lucky bonus.

## Features
- Login system using username and PIN
- Check current account balance
- Deposit money
- Withdraw money with validation
- Lucky bonus generator
- Exit option
- Input validation and error handling

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone this repository.
2. Open your HTML file in a browser.
